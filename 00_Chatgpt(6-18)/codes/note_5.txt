## âœ” Example 1 â€” Basic Linear Search

```java
public class LinearSearch {
    static int linearSearch(int[] arr, int target) {

        for(int i = 0; i < arr.length; i++) {
            if(arr[i] == target) {
                return i; // return index
            }
        }
        return -1; // not found
    }

    public static void main(String[] args) {
        int[] arr = {5, 8, 12, 3, 9};

        int index = linearSearch(arr, 12);
        System.out.println("Found at index: " + index);
    }
}
```

### **OUTPUT**

```
Found at index: 2
```

# ðŸŸ© **2. BINARY SEARCH (Basic â†’ Deep)**

ðŸ‘‰ Works only on **sorted arrays**
ðŸ‘‰ Time Complexity = **O(log n)**
ðŸ‘‰ Uses **divide & conquer**

---

## âœ” Example 1 â€” Iterative Binary Search

```java
public class BinarySearch {

    static int binarySearch(int[] arr, int target) {

        int start = 0;
        int end = arr.length - 1;

        while(start <= end) {

            int mid = start + (end - start) / 2;

            if(arr[mid] == target) {
                return mid; // found
            }
            else if(arr[mid] < target) {
                start = mid + 1; // go right
            }
            else {
                end = mid - 1; // go left
            }
        }
        return -1;
    }

    public static void main(String[] args) {
        int[] arr = {2, 5, 8, 12, 16, 23};

        System.out.println(binarySearch(arr, 12));
    }
}
```

### **OUTPUT**

```
3
```

## âœ” Example 2 â€” Recursive Binary Search

```java
public class BinaryRecursive {

    static int search(int[] arr, int start, int end, int target) {

        if(start > end) return -1;

        int mid = start + (end - start) / 2;

        if(arr[mid] == target) return mid;

        if(target < arr[mid])
            return search(arr, start, mid - 1, target);
        else
            return search(arr, mid + 1, end, target);
    }

    public static void main(String[] args) {

        int[] arr = {3, 6, 9, 12, 15};
        int index = search(arr, 0, arr.length - 1, 15);

        System.out.println(index);
    }
}
```

### **OUTPUT**

```
4
```

# ðŸŸ© **3. TRAPPING RAINWATER Problem (Deep Explanation + Code)**

ðŸ‘‰ Hard-level problem
ðŸ‘‰ Water stored depends on:
**leftMax[i] = max height left side**
**rightMax[i] = max height right side**

Water at index =
`min(leftMax, rightMax) â€“ height[i]`

Time Complexity = **O(n)**
Space = **O(n)**

---

## âœ” Example â€” Trapping Rainwater

```java
public class RainWater {

    static int trap(int[] height) {

        int n = height.length;

        int[] leftMax = new int[n];
        int[] rightMax = new int[n];

        // 1. Build leftMax
        leftMax[0] = height[0];
        for(int i = 1; i < n; i++) {
            leftMax[i] = Math.max(leftMax[i - 1], height[i]);
        }

        // 2. Build rightMax
        rightMax[n - 1] = height[n - 1];
        for(int i = n - 2; i >= 0; i--) {
            rightMax[i] = Math.max(rightMax[i + 1], height[i]);
        }

        // 3. Water trapped
        int water = 0;
        for(int i = 0; i < n; i++) {
            water += Math.min(leftMax[i], rightMax[i]) - height[i];
        }

        return water;
    }

    public static void main(String[] args) {
        int[] arr = {4, 2, 0, 3, 2, 5};

        System.out.println("Total trapped water = " + trap(arr));
    }
}
```

### **OUTPUT**

```
Total trapped water = 9
```
