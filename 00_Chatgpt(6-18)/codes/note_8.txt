# ‚úÖ **1. STRING PROPERTIES (VERY IMPORTANT)**

## ‚úî **Property 1 ‚Äî Strings are IMMUTABLE**

Once created, you **cannot change** a string.

### Example:

```java
public class StringImmutable {
    public static void main(String[] args) {

        String s = "Hello";
        s.concat(" World");   // not added (ignored)
        System.out.println(s);

        s = s.concat(" World"); // creates new object
        System.out.println(s);
    }
}
```

### OUTPUT

```
Hello
Hello World
```

---

## ‚úî **Property 2 ‚Äî Stored in the String Constant Pool**

```java
String a = "Java";
String b = "Java";  // same object from pool
```

### Example

```java
public class StringPool {
    public static void main(String[] args) {
        String a = "Java";
        String b = "Java";

        System.out.println(a == b); // true (same object)
    }
}
```

### OUTPUT

```
true
```

---

## ‚úî **Property 3 ‚Äî Using new creates a NEW object**

```java
String a = new String("Java");
String b = new String("Java");
```

### Example

```java
public class StringNew {
    public static void main(String[] args) {
        String a = new String("Java");
        String b = new String("Java");

        System.out.println(a == b); // false (different objects)
    }
}
```

### OUTPUT

```
false
```
